[b]Terracrafted[/b]
[i]A Minecraft-style crafting and gameplay enhancement mod for Terraria.[/i]

[hr][/hr]
[b]📖 About[/b]
This mod currently provides a grid-based crafting system, with plans to add more Minecraft-inspired mechanics in the future, aiming to deliver a different experience for obtaining and creating items compared to vanilla.

The mod is a complete rewrite and expansion of its predecessor, [b]MinecraftWorkbench[/b] from the 1.4 Alpha era.

[quote][b]⚠️ Note[/b]
The mod is in early development, and recipes are not yet complete. Please enable the relevant option in the config file when trying it out.[/quote]

[b]✨ Current Features[/b]
[list]
[*][b]Grid Crafting System[/b]
After enabling "Use Overhaul Gameplay" in the config, you can freely arrange materials in the crafting grid to create items. The Guide NPC will also display the corresponding recipes.
[*][b]Smelting System[/b]
Smelting system is now added. (Currently NOT tested in multiplayer mode!)

[/list]

*This mod supports loading custom recipes or modifying existing ones from external JSON files. Please check the mod's configuration directory (.../My Games/Terraria/tModLoader/TerraCraft/). Template files will be automatically generated when the folder is empty on first run.
